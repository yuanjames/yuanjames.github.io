Paper: Iterative Deep Graph Learning for Graph Neural Networks: Better and Robust Node Embeddings
The paper Github is https://github.com/hugochan/IDGL.

The main ideas used in this paper has four:

        1. Adjacency Matrix is no longer from spectral domain, it is learnd based on nodes' similarrity.
        2. Achor-Based: sampling. 
        3. Node-anchor message passing.
        4. Joint Learning with A Hybrid Loss
